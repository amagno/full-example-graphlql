### Executanto
> - Certifique-se de ter o [Node.js](https://nodejs.org/) 6.11.0 ou superior instalado
> - Instale o [Yarn](https://yarnpkg.com/)
> - Instale o MongoDB localmente ou Utilize o Docker para roda o mesmo [Docker](https://hub.docker.com/_/mongo/)
> - Caso queira mude o endere√ßo do Servidor Mongo [MONGO_URL](https://github.com/amagno/full-example-graphlql/blob/master/nodemon.json) no arquivo [nodemon.js](https://github.com/amagno/full-example-graphlql/blob/master/nodemon.json) 
```shell
yarn install
yarn watch
```
